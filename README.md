# Clustar UI

Astro-first UI components.

## Install

```bash
npm install clustarui
```

## Usage

### Named import

```astro
---
import { Button } from "clustarui";
---

<Button variant="solid">Hello</Button>
```

### Direct path import

```astro
---
import Button from "clustarui/Button.astro";
---

<Button>Hi</Button>
```

## Status

Early minimal release to establish the package name and public API.
